
TÀI LIỆU PHÂN TÍCH THIẾT KẾ HỆ THỐNG QUẢN LÝ TRUNG TÂM STEM
Công nghệ: C# ASP.NET Core MVC 10, Entity Framework Core, SQL Server.
Mô hình: MVC, Repository Pattern, N-Tier Architecture.
Lưu trữ File &amp; Tài liệu (Chi phí 0đ): YouTube API (Unlisted) cho Video / Google Drive cho Ảnh / Google Slides (Iframe) cho Giáo án. AI Integration (Miễn phí): Google AI Studio API (Gemini 1.5 Flash) phục vụ sinh nội dung và chấm điểm kỹ năng.
1. Sơ đồ ngữ cảnh (Context Diagram)
Thể hiện hệ thống ở trung tâm và các tác nhân tương tác xung quanh.
Hệ thống trung tâm: Hệ thống Quản lý Trung tâm STEM.
Các Tác nhân và Luồng dữ liệu (Data Flow):
Admin (Quản trị viên): Quản lý cấu hình, xem xét minh chứng giảng dạy (hình ảnh/video), đọc nhận xét, quản lý nội dung web (CMS).
Teacher (Giáo viên): Nhập điểm danh, tải lên minh chứng sản phẩm học sinh, nhập ý chính để AI tự động tạo lời nhận xét mượt mà.
Student (Tài khoản Phụ huynh/Học sinh): Xem lịch học, xem hình ảnh/video sản phẩm con làm hôm nay, đọc nhận xét chuyên nghiệp từ giáo viên.
LLM API (ChatGPT/Gemini - External): Nhận input gạch đầu dòng của giáo viên -&gt; Trả về đoạn văn nhận xét trau chuốt.
Cloud Storage (S3/R2/Drive - External): Nhận file hình ảnh/video -&gt; Trả về URL.

3. Biểu đồ Use Case
Gom nhóm theo các tác nhân chính trong hệ thống.
Teacher: Đăng nhập, Xem lịch dạy, Mở tài liệu giáo án (Iframe), Điểm danh, Upload ảnh/video thực hành, Dùng AI tạo nhận xét, Mượn/Trả thiết bị STEM.
Admin: Quản lý tài khoản, Xếp lịch học, Phân công giáo viên, Quản lý kho tổng, Quản lý thu chi/hóa đơn, Xem báo cáo minh chứng, Đăng tin tức CMS.
Student/Parent: Đăng nhập, Xem lịch học, Xem thư viện ảnh sản phẩm của con, Đọc đánh giá buổi học, Theo dõi hóa đơn học phí.
